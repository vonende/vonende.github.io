Allgemeines

Was ist Openboard?

Openboard ist eine kostenlose, freie und interaktive Whiteboard-Software, die speziell für den Unterricht verwendet wird. Mit diesem Programm können Sie interaktive Displays wie Smartboards oder Tablets gezielt in den Unterricht integrieren. Die Software steht in den Sprachen Deutsch, Englisch, Französisch und Italienisch zur Verfügung.
Download für private Endgeräte: https://openboard.ch/download.de.html

Weshalb lohnt es sich, mich mit Openboard zu beschäftigen?

In unserer Schule sind nun bereits digitale Whiteboards drei verschiedener Hersteller in Benutzung. Die Verwendung der gewohnten Smart-Notebook-Software auf den Geräten anderer Hersteller erfordert eine sehr teure Lizenz. Das Geld würde an anderer Stelle schmerzlich fehlen. Damit unabhängig von der Hardware eine einheitliches Arbeiten an den digitalen Whiteboards möglich ist, wird auf jedem Gerät die voll funktionsfähige Whiteboard-Software Openboard zur Verfügung gestellt. Die Benutzung ist freiwillig und lediglich ein Angebot.

Vorteile

  • Die Software ist kostenlos und ohne Registrierung nutzbar und mit folgenden Betriebssystemen kompatibel: Windows, MacOS, Linux
  • Zur Nutzung ist keine Internetverbindung notwendig.
  • Anwendung auf dem Notebook/Tablet zusätzlich mit Touch- und Stifteingabe möglich
  • Whiteboard-Software läuft auf allen gängigen Smartboards, digitalen Tafeln, elektronischen Whiteboards usw.
  • erstellte Dokumente können im Openboard-Dateiformat oder als PDF an Lehrerinnen und Lehrer verteilt werden
  • Unterrichtssoftware kann auch auf dem heimischen Computer installiert werden und ermöglicht Unterrichtsvorbereitung
  • viele integrierte Werkzeuge: Stift, Radiergummi, Marker, LaserPointer, Uhr, Timer, Lineal, Zirkel, Lupe, Taschenrechner usw.

Nachteile

  • Die automatische Stifterkennung der Whiteboards der Firma Smart funktioniert in Openboard nicht.
  • Ein Import der Smart-Notebook-Dateien ist nicht möglich. Man kann Inhalte aus Smart-Notebook als Bild oder PDF exportieren und diese anschließend in Openboard importieren. Sie verlieren dann aber ihre Interaktivität.

Installation

Vorbereitungen

Damit Apps aus dem Unternehmensportal installiert werden können, muss sich Windows auf einem halbwegs aktuellen Stand befinden. Wie Sie den Stand der Updates einsehen und Windows aktualisieren können, erfahren Sie in diesem Abschnitt.

Im Startmenü finden Sie links ein Zahnrad:

Dieses Zahnrad führt Sie zu den Einstellungen:

In den Einstellungen finden Sie die Kategorie "Update & Sicherheit":

Wenn Sie ihr Gerät regelmäßig nutzen, wird Windows vermutlich im Hintergrund die neuesten Updates eingespielt haben:

Aber auch wenn Ihnen Windows mitteilt, sich auf dem neuesten Stand zu befinden, lohnt es sich, nach Updates zu suchen:

Tatsächlich findet sich oft ein installierbares Update:

Achtung

Sogenannte kumulierte Sicherheitsupdates oder Versionsupdates sind oft sehr groß und können teilweise sehr lange dauern. Sie sollten zudem mit mehreren Neustarts rechnen. Stoßen Sie deshalb kein Update an, während Sie mit dem Gerät produktiv arbeiten.

Tipp

Aktualisieren Sie ihr Gerät am Besten, wenn Sie daheim putzen oder fernsehen. So können Sie den Aktualisierungsprozess im Blick behalten und erforderliche Neustarts durchführen.

Anpassen der Schrift- und Symbolgrößen

Wenn Sie Openboard nun zum ersten Mal starten, dann werden Sie feststellen, dass alle Symbole und die Schrift sehr klein sind:

Beenden Sie Openboard wieder, indem Sie ganz oben rechts auf "Openboard" klicken und in der Liste ganz unten "Beenden" auswählen.

Gehen Sie nun mit dem Mauszeiger zum Openboard-Symbol unten in der Taskleiste. Klicken Sie nun mit der rechten Maustaste auf das Symbol oder mit 2 Fingern auf das Touchpad, dann noch einmal auf "OpenBoard" und danach auf "Eigenschaften":

Wählen Sie die Karte "Kompatibilität" aus und klicken Sie auf "Hohe DPI-Einstellungen ändern":

Wählen Sie nun im unteren Auswahlbereich "System" aus. Schließen Sie anschließend beide Fenster, indem Sie jeweils auf "OK" klicken.

Wenn Sie jetzt noch einmal Openboard starten, sollte alles viel besser erkennbar sein:
